# 🛒 LETSCART

**Shop Smarter, Live Better**

LETSCART is a modern AI-ready e-commerce platform built using **Next.js**, **React**, and **Tailwind CSS**. The project focuses on delivering a fast, responsive, and user-friendly shopping experience with a scalable architecture that is ready for future backend, AI, and payment integrations.

---

## 📖 Overview

LETSCART is designed as a production-ready e-commerce application with a clean UI, reusable components, and modern frontend architecture. The current version focuses on the frontend, while future phases will introduce authentication, backend APIs, AI-powered search, and secure payment processing.

## 🛠️ Installation

Clone the repository

```bash
git clone https://github.com/Aayushkrprasad/LetsCart-SummerIntership.git
```

Go to the project folder

```bash
cd LetsCart-SummerIntership
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
